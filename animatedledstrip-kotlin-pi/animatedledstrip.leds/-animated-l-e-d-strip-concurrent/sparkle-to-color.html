sparkleToColor

fun sparkleToColor(destinationColor: ColorContainer, delay: Int = 50, delayMod: Double = 1.0, concurrent: Boolean = true): Unit

A non-repetitive function to run a Sparkle To Color animation.

Very similar to the Sparkle animation, but the LEDs are not reverted to their original color after the sparkle.

If concurrent = true (default), a separate thread will be created for each pixel. Each thread waits for 0-5 seconds before sparkling its pixel.

If concurrent = false, shuffleArray is shuffled and used to determine the order in which the LEDs are sparkled. Unlike the concurrent version, only one pixel will sparkle at any given time.

Parameters

destinationColor - The color the pixels will sparkle with

delay - Duration of each sparkle

delayMod - Multiplier for delay

concurrent - Use concurrent sparkle algorithm?
