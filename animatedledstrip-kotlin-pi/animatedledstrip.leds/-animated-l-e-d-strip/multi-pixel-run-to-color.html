multiPixelRunToColor

fun multiPixelRunToColor(spacing: Int, chaseDirection: Direction, destinationColor: ColorContainer, delay: Int = 150, delayMod: Double = 1.0): Unit

A non-repetitive function to run a Multi Pixel Run To Color animation.

Parameters

spacing - Spacing between lit leds (for example, if spacing is 3 and led 0 is lit, led 3 will also be lit)

chaseDirection - Direction of animation

destinationColor - Color of moving pixels and color strip will be at end of animation

delay - Delay between moves

delayMod - Multiplier for delay
