multiPixelRun

fun multiPixelRun(spacing: Int, chaseDirection: Direction, colorValues1: ColorContainer, colorValues2: ColorContainer = CCBlack): Unit

Function to run a Multi Pixel Run animation.

Similar to Pixel Run but with multiple leds at a specified spacing.

Parameters

spacing - Spacing between lit leds (for example, if spacing is 3 and led 0 is lit, led 3 will also be lit)

chaseDirection - Direction of animation

colorValues1 - Color of moving pixels

colorValues2 - Color of background pixels
