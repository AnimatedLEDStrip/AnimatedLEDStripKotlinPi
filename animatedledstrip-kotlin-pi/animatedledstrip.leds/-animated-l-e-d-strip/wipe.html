wipe

fun wipe(colorValues: ColorContainer, wipeDirection: Direction, delay: Int = 10, delayMod: Double = 1.0, startPixel: Int = 0, endPixel: Int = numLEDs - 1): Unit

A non-repetitive function to run a Wipe animation.

Similar to a Pixel Run animation, but the pixels do not revert to their original color.

Parameters

colorValues - Color of moving pixel and color strip will be at end of animation

wipeDirection - Direction of animation

delay - Delay between moves

delayMod - Multiplier for delay
