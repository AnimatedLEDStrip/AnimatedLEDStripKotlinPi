sparkleToColor

fun sparkleToColor(destinationColor: ColorContainer, delay: Int = 50): Unit

A non-repetitive function to run a Sparkle To Color animation.

Very similar to the Sparkle animation, but the LEDs are not reverted to their original color after the sparkle.

shuffleArray is shuffled and used to determine the order in which the LEDs are sparkled. Only one pixel will sparkle at any given time.

Parameters

destinationColor - The color the pixels will sparkle with

delay - Duration of each sparkle
